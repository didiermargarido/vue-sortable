# Vue Sortable

Easily add drag-and-drop sorting to your Vue.js application using **sortablejs** library.

## Demo

https://didiermargarido.github.io/vue-sortable/

## Installation
```
npm install
npm run serve
```

### Lint
```
npm run lint
```

### Documentation
